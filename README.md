# SuperFB

Adalah tools crack super cepat dalam hitungan detik

# Simple Usage

$ pkg update && pkg upgrade

$ pkg install git

$ pkg install python

$ pip install bs4

$ pip install mechanize

$ pip install requests

$ git clone https://github.com/KangProf/superFB

$ cd superFB

$ ls

$ cd lib

$ python super.py
